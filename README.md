# mean-app

Requirement:
For MongoDB version:
1. Node.js
2. Express
3. Mongoose
4. MongoDB
5. body-parser
6. morgan

For Squelize version:
1. Node.js
2. Express
3. Sequelize
4. Postgres
5. body-parser
6. morgan

Sequelize generating model:
In terminal run:
$sequelize model:generate --name SubjectArea --attributes 'subject:string, active:boolean' --underscored true
